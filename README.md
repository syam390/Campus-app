Welcome to CampusSync - your all-in-one campus super-app! This application is designed to streamline campus life for students, providing easy access to events, attendance tracking, placement preparation, and more.

Events & Feed: Stay updated with campus happenings. Get personalized event recommendations and view your own calendar.
Attendance Tracker: Mark your attendance with a quick face scan.
Eco Tracker: Calculate your carbon footprint and compete with other hostels on the eco-friendly leaderboard.
Placement Prep: Get your resume analyzed by AI, complete with an ATS score, constructive "roasts," and company-specific interview questions.
This is a Next.js project bootstrapped with create-next-app.

To get a local copy up and running, follow these simple steps.

You need to have Node.js and npm installed on your machine.

Clone the repo
git clone https://github.com/syam390/Campus-app.git

Install NPM packages
npm install

Run the development server:
npm run dev

Open http://localhost:9002 with your browser to see the result.
